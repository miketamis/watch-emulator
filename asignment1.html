The Basic Idea

The idea is simple, a smart watch app that tracks how many sit ups you do daily and motivates you towards completing your set goal.

Similar apps

I did some research on existing apps and found several apps for both Android and iOS, but none for any smart watch. The Android apps include:

App 1: Sit Ups

Very basic application.

Initial Setup:

When the user starts the application, it asks to select one of four goals: 100, 150, 200 and 250 sit ups in a row. Then it requires the user to do as many sit ups as they can in one set so it can determine the user’s start position. It also asks if the user is male or female.

How goals are set:

The goals are set manually by the user when they start the application for the first time. Once you have completed the goal, it puts you in strength training mode where it just repeats the same group of sets each day and no longer pushes you to improve yourself. It says that if you want you can reset the application and set a new goal but this means you lose your log :(

Daily use:

Each day when you go to use the application it presents you with the day’s workout. This consist of about 4 sets of sit ups. Once started it takes you through each set stating the number of sit ups you have to do for each set. In between each set it has a cool-down timer before the next set starts.

After completing all of the sets it asks how you how you found these sets so it can give you an appropriate number of reps next time. This is done by selected one of three options "It was too easy.", "It was just right." and "It was too hard."

Then the user is presented with a progress screen where they can see how close they are to achieve their goal.

Achievement System:

Very basic, you can get three different trophy badges that you can post to your Facebook wall.

  1. Bronze: when your 1/3 of the way to your goal.
  2. Silver: when your 2/3 of the way to your goal.
  3. Gold: when you have achieved your goal.

Data:

This app collects how many reps you do in each workout and lets you view it as list.

App 2: Abs Sit Ups Workout

Feature packed app. This app has lots of sounds. Has the ability to notify you daily/every x days to do your workout.

Initial Setup:

Just requires you todo as many sit ups as you can. Doesn't require you to set goals or specify your gender.

How goals are set:

You start out at level 1 (or whatever level it puts you at based on your initial sit up count).
At the end of:
level 1: you should be able todo 100 straight sit ups.
Level 2: 150 straight sit ups.
Level 3: etc....

Each level is broken down to 3 sub levels. Each sub level has a pre-set workout for you each day.

Daily use:

Very similar to App 1. “Sit Ups”. Alternates between telling you how many sit ups to do and then a cool down. It has a cool feature that while you are doing the sit ups you can tap the screen to cause the number to go down instead of counting the reps yourself.

At the end of the work out it asks you the same three questions as App 1: "It was too easy.", "It was just right." and "It was too hard." Too easy cause you to go up a sublevel, just right keeps you on your current sublevel, too hard drops you down a sublevel.

Achievement System:

Doesn't have one.

Data:

Has the ability to sync with Google Fit. Although all it synchronises is calories burnt, it doesn't sync reps etc.

Has ability to see the number of reps completed and Calories burnt on a graph.

Allows you to back-up the data to SD card and Dropbox

App Plan

I'm going to use the workout schedule found on twohundredsitups.com

The following sketch shows the rough app flow:

When you open the app for the very first time you are presented with the "Welcome" screen. Once the user hits the middle button they are taken to the "age" selection menu.

The "age" selection screen gives you three options which correspond to the ages in twohundredsitups.com initial test. Clicking the top button sets the age to "under 30", clicking middle button sets the age to "30 to 39" and clicking bottom button set the age to "40 plus". It then take you to the "initial amount" test.

The "initial amount" screen wants you to do as many sit ups as you can, and then enter the number, the top button increase the amount, the bottom button decrease the amount. The middle button sets the amount and takes you to the main "menu".

The main menu is where you can find everything. T0 begin with all we need is “settings” and “start”. When you select “start” it will take you to the sit up screen. To get back to the main menu at any stage you can hold down the Back Button

The "sit ups" screen will indicate how many sit ups you must perform, the progress bar at the top will show where you are at, dots indicate sit ups and arrows indicate rests. Once the user has finished all their sit ups they hit the middle button to move on, which takes them to the "rest" screen.

The "rest" screen has how many seconds left you have to rest for. It will also have a pie chart in the background to give some visual feedback. If the user decides they don't want to rest the can skip the rest by clicking the middle button.

Once you have completed all of your sets, you will be presented with the "done" screen, where you get three options, if you select "Too easy" you go up an exercise level, if you select "Just right" you stay on the current exercise, if you chose "Too hard" you go down an exercise level.

Module Loader/Packers

I will want to use a module loader so I can separate my code into multiple files to keep my code cleaner.

When looking at module loaders, you have to make a choice of which module format you want, AMD or CommonJS.

AMD is what RequireJS uses and CommonJS is what nodeJS uses. I have used both nodeJS and RequireJS a lot in different projects, and I prefer CommonJS because you don't have to wrap your module in a define function, and you don't get confused by which variables equal which modules, and the code looks cleaner (less indents which I like).
